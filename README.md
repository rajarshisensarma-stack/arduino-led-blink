# Arduino LED Blink Projects

## 📌 Project Description
This repository contains basic Arduino LED blinking programs.

## 🔧 Components Used
- Arduino Uno
- LED
- Resistor
- Jumper wires

## 💡 Project 1: Basic Blink
LED turns ON and OFF every 1 second.

## 💡 Project 2: Alternate Blink
Two LEDs blink alternately.

## 🧠 What I Learned
- Digital output using Arduino
- Basic timing using delay()
